# mi-primer-codespace

Repositorio con ejercicios básicos de Python orientados a estudiantes universitarios que están iniciando en la programación. Los ejemplos cubren conceptos fundamentales como variables, estructuras de control, funciones, listas y manejo básico de archivos.

---

## 📖 Descripción

Este repositorio tiene como objetivo servir como material de apoyo para cursos introductorios de programación en Python. Los ejercicios están diseñados para desarrollar habilidades de pensamiento lógico y resolución de problemas mediante ejemplos prácticos.

### Temas incluidos

| Tema                       | Descripción                                    |
| -------------------------- | ---------------------------------------------- |
| Variables y tipos de datos | Uso de enteros, flotantes, cadenas y booleanos |
| Operadores                 | Operaciones aritméticas y lógicas              |
| Estructuras de control     | Condicionales (`if`, `elif`, `else`)           |
| Ciclos                     | Bucles `for` y `while`                         |
| Funciones                  | Definición y uso de funciones                  |
| Listas y diccionarios      | Manipulación de estructuras de datos           |
| Archivos                   | Lectura y escritura básica de archivos         |

---

## 🔧 Prerrequisitos

Antes de comenzar, asegúrate de tener instalado:

| Software                      | Versión recomendada    |
| ----------------------------- | ---------------------- |
| Python                        | 3.10 o superior        |
| Git                           | Última versión estable |
| Visual Studio Code (opcional) | Última versión         |

Verifica la instalación de Python ejecutando:

```bash
python --version
```

o

```bash
python3 --version
```

## ▶️ Cómo ejecutar

1. Clona el repositorio:

```bash
git clone https://github.com/usuario/mi-primer-codespace.git
```

2. Accede al directorio del proyecto:

```bash
cd mi-primer-codespace
```

3. Ejecuta cualquiera de los ejercicios:

```bash
python ejercicios/ejercicio_01_variables.py
```

o, dependiendo de tu sistema:

```bash
python3 ejercicios/ejercicio_01_variables.py
```
